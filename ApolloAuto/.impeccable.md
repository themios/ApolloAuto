## Design Context

### Users

**Who:** Budget-conscious used-car buyers in Ventura County and Los Angeles County—Simi Valley, El Monte, Thousand Oaks, San Gabriel Valley, and surrounding areas.

**Context:** Often on mobile, researching before visiting a lot. Many are first-time buyers, rebuilding credit, or have past repossession/bankruptcy. They've been burned by hidden fees or impersonal dealers and want to talk to a real person (Tim), not a call center.

**Job to be done:**
1. Find a trustworthy local dealer with financing options
2. Understand what to expect (financing, paperwork, trade-ins) before visiting
3. Contact Tim—call, text, or form—and/or browse lot inventory on external sites

**Primary conversions:** Phone/text to Tim · Contact form · Click-through to lot inventory · Read FAQ/guides (trust-building)

### Brand Personality

**Voice:** First-person plural (“we”), Tim by name, plain English, no dealer hype. Acknowledge bad credit without shame. Never promise guaranteed approval.

**Three words:** Trustworthy · Warm · Direct

**Emotional goals:** Confidence (this dealer is honest) · Relief (financing is possible) · Approachability (Tim is reachable, same-day replies)

**Positioning:** Family-owned Southern California dealer—not a national chain, not a generic car lot. Transparent pricing, no add-on fees, real redacted contract PDFs as proof of expertise.

**Words to use:** used cars, pre-owned, family-owned, transparent pricing, no hidden fees, financing, trade-in, Ventura County, Los Angeles County, Simi Valley, El Monte

**Words to avoid:** “Best deals!”, “Act now!”, “Guaranteed approval”, corporate jargon (synergy, solutions, leverage), keyword stuffing

**Creative latitude:** Tim is **open to something new and exciting**—not locked to the current Golden Coast Trust look forever. Evolution is welcome when it stays trustworthy and local. All visual experiments must be **reversible** (see Reversibility below).

### Aesthetic Direction

**Baseline concept:** **Golden Coast Trust** — credible teal-navy with California gold warmth on cream paper. Editorial, human, not a flashy dealership template.

**Evolution target:** Blend Golden Coast warmth with editorial confidence from Tim’s reference sites—calm like [Daylight](https://daylightcomputer.com/), structured like [ISI Global](https://www.isiglobal.com/), clear like [Mergado](https://www.mergado.com/), pathway-driven like [Ever Cars](https://www.evercars.com/), specialist credibility like [Bring a Trailer Sell](https://sell.bringatrailer.com/)—without copying their palettes or falling into generic EV-startup / SaaS templates.

**Theme:** Light mode (buyers research on phones during the day). Cream/warm paper surfaces, not dark dashboards.

**Typography:**
| Role | Craft v2 default | Legacy (`html.theme-legacy`) | Future |
|------|------------------|------------------------------|--------|
| Display / H1–H2 | Sora (500–800) | Outfit | Admin-configurable via CMS theme settings |
| Body / UI | Karla (400–700) | Albert Sans | Admin-configurable via CMS theme settings |

Fonts load from `css/styles.css` (`@import`); subpages use `theme-presets.css` only — no per-page Google Font links.

Fluid `clamp()` headings on marketing pages. Hero lead copy capped ~48–65ch. Font changes must remain WCAG-readable and load performantly (preconnect + subset weights).

**Color tokens** (craft v2 defaults in `css/styles.css` `:root`—CMS should override via CSS variables, not hard-coded hex in HTML):

| Token | Use |
|-------|-----|
| `--navy` | Headings, footer, primary buttons |
| `--navy-soft` | Hover, gradients, trust checkmarks (`--success` aliases this) |
| `--gold` | Accents, eyebrows, CTA highlights, focus rings |
| `--gold-light` | Quote attribution, hover |
| `--paper` | Page background |
| `--ink` | Body text |

**Imagery:** Stock photography acceptable for now; swap to real lot/customer photos later without layout changes (use consistent aspect ratios and `object-fit` patterns).

**Social proof:** **Prominent review/testimonial carousel** on homepage (and optionally location pages)—not subtle. Structure inspired by Ever + Mergado testimonial bands; content managed via admin.

**Key UI patterns (current + planned):**
- Logo mark — gradient badge with stylized window shape
- Hero — gradient + grain, kicker pill, trust bar, rotating gallery, contact panel
- **Prominent testimonial carousel** (live — JSON + static fallback; admin CMS planned)
- **Rotating hero/media gallery** (live via `data/hero-slides.json`; admin upload planned)
- Location spotlight cards — top-of-page lot CTAs
- Sticky gold “Call Tim” pill on mobile
- Scroll-triggered `.reveal` animations; card hover lift; `prefers-reduced-motion` respected
- Global `:focus-visible` gold ring on links, buttons, and tiles

### Anti-references (explicit)

- Generic “AI dealership” aesthetics (purple gradients, neon-on-dark, cookie-cutter templates)
- Corporate chain dealer sites (pressure tactics, stock hero sliders with “ACT NOW”)
- National aggregators (CarGurus-style clutter as the brand hub)
- Inter, Roboto, system-default typography monoculture
- Keyword-stuffed unreadable SEO copy
- **ApolloAuto-AIStudio** visual direction (Montserrat + orange/blue)—do not merge into main site

### Design Principles

1. **Trust before flash** — Exciting design must still answer “would a skeptical buyer feel safe here?”
2. **Tim is the product** — Human contact beats chatbots; carousel and content support trust, not distraction from calling/texting.
3. **Local, not national** — Two distinct lots; copy and schema signal Ventura & LA counties.
4. **Plain English, editorial rhythm** — Short sentences, varied spacing; no dealer hype.
5. **SEO serves the reader** — Authority content earns rank; keywords natural in H1, meta, opening paragraph.
6. **Reversible by default** — Theme tokens, fonts, colors, and CMS-driven content live in data/CSS variables + git history. No one-way paint-over of static HTML without a rollback path (preserve prior `styles.css` snapshot or theme preset in admin).
7. **Admin owns the surface** — Blogs, news, rotating images, testimonials, and visual theme should be editable without redeploying HTML (extend existing `/admin/` + SQLite API).
8. **Lightweight & fast** — Static HTML delivery; LCP target < 2.5s mobile even with carousel and stock images (lazy load, responsive src).
9. **Accessibility — WCAG 2.1 AA best effort** — Skip links, semantic HTML, focus states, color contrast on all theme presets, reduced motion, carousel keyboard/ARIA patterns, alt text on all stock photos.

### Impeccable Alignment Notes

When applying impeccable craft to this project:

- Replace legacy **border-left accent stripes** in `styles.css` when redesigning those components (use background tints, full borders, or structural emphasis).
- New font picks must follow impeccable font selection (avoid reflex monoculture fonts); CMS font picker should curate allowed pairs.
- Prefer **OKLCH** for new theme tokens; migrate from hex when touching the palette.
- No purple/cyan AI gradients, gradient text, glassmorphism-as-decoration, or hero-metric templates.
- Carousel: autoplay optional with pause control; respect `prefers-reduced-motion` (static first slide or no autoplay).
