# Dealership photos

Replace these placeholder images with your own from Google Business:

1. Search [Apollo Auto Simi Valley on Google](https://www.google.com/search?q=apollo+auto+simi+valley)
2. Open your business listing → Photos
3. Download your favorites and save here using these names:

| File | Suggested use |
|------|----------------|
| `hero-car-alt.jpg` | Hero background (wide, ~1200×700) |
| `lot-1.jpg` | Simi Valley location card |
| `lot-2.jpg` | El Monte location card |
| `lot-3.jpg` | Gallery |

Keep filenames the same so the site updates automatically.
